previewConfig
Type: PreviewConfigoptional

Optional object used to configure PreviewConfig.

Interface for configuring previews, including the number of visible previews, navigation arrows, and sizes.

Default value:

{ visible: true, number: 4, arrows: true, clickable: true, width: 100 / 4 + '%', maxHeight: '200px', breakpoints: { xSmall: 100, small: 100, medium: 150, large: 200, xLarge: 200 } }
dotsConfig
Type: DotsConfigoptional

Optional object used to configure DotsConfig.

Default value:

slideConfig
Type: SlideConfigoptional

Optional object used to configure SlideConfig to manage sliding features of modal-gallery.

Enables or disables infinite sliding and configures side-previews and autoPlay behavior. When disabled, navigation is restricted to within the first and last image boundaries. By default, infinite sliding is disabled, side-previews are visible with width: 100px and height: auto, and autoPlay is disabled.

Default value:

{ infinite: false, playConfig: { autoPlay: true, interval: 5000, pauseOnHover: true }, sidePreviews: { show: true, size: { width: '100px', height: 'auto' } } }
