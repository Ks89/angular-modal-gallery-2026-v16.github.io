Download Advanced example

Modal gallery example with downloadable images and download button.


Warning: For security reasons, Google Chrome and other modern browsers block cross-origin downloads. To use the download feature in this example, images must be hosted on the same server as your Angular application. Otherwise, the browser will open the image in a new tab instead of downloading it automatically. For more information, see HERE
