Supported Browsers

As of version 9.0.0, Internet Explorer 11 and legacy Microsoft Edge (pre-Chromium) are no longer supported.


Installation

Based on the table above, install the required dependencies.

For the latest version of @ks89/angular-modal-gallery run

  • npm i --save @ks89/angular-modal-gallery@latest
  • npm i --save @angular/cdk

If you require angular-modal-gallery version 5 or 3, use the legacy npm package angular-modal-gallery (without the @ks89 scope).

Default icons do not require Font Awesome. Starting from angular-modal-gallery 5.0.0, Font Awesome is optional. To use custom button icons, you may provide your own CSS or use Font Awesome (versions 4 or 5). Font Awesome 5 is officially supported from angular-modal-gallery 5.3.0 and later.


Update your project

After installing the dependencies, follow the steps below.

These instructions apply to Angular CLI projects. If you are using @angular/universal, please refer to the official examples.


Step 1: Import dependencies

Standalone components

If your project uses standalone components, import GalleryModule via importProvidersFrom and pass appConfig to bootstrapApplication:

        bootstrapApplication(AppComponent, appConfig)
          .catch((err) => console.error(err));

Then configure importProvidersFrom to load GalleryModule as follows:

        import { GalleryModule } from '@ks89/angular-modal-gallery';

        export const appConfig: ApplicationConfig = {
          providers: [
            provideBrowserGlobalErrorListeners(),
            provideZoneChangeDetection({ eventCoalescing: true }),
            provideRouter(routes),
            importProvidersFrom(
              // import GalleryModule. Install @ks89/angular-modal-gallery first
              GalleryModule
            )
          ]
        };

NgModule-based projects

For NgModule-based projects, import GalleryModule in your root module and in every module that uses this library:

        import {GalleryModule} from '@ks89/angular-modal-gallery';
        @NgModule({
          imports: [
            GalleryModule,
            ...],
          declarations: [ AppComponent, ... ],
          providers: [ ... ],
          bootstrap: [ AppComponent ]
        })
        export class AppModule {}

Step 2: Update global CSS

Add the following styles to your global (CSS/SCSS/LESS) file:

        @use "~@angular/cdk/overlay-prebuilt.css" as *;
        .ks-modal-gallery-backdrop {
                  background: #000 !important;;
                  opacity: 0.85 !important;;
        }
        .ks-modal-gallery-panel {
                  z-index: 90000 !important;
        }

Note: @import is deprecated. Always use @use ... as *. For more details, refer to the official documentation


Step 3: (optional) update tsconfig file

If needed, add @types to tsconfig.json:

        {
          ...
          "compilerOptions": {
            ...
            "typeRoots": ["node_modules/@types"]
          }
        }
