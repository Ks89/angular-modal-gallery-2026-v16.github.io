UpperButtons

The UpperButtons component renders action buttons configured via ButtonsConfig.


ButtonsConfig

Interface to customize modal upper buttons.


visible
Type: boolean

If false, all buttons are hidden, otherwise not. true by default.
strategy
Type: ButtonsStrategy

Enum controlling which buttons are visible, based on predefined patterns (default, simple, advanced, etc.). Defaults to ButtonsStrategy.DEFAULT.
buttons
Type: ButtonConfig[]optional

When using ButtonsStrategy.CUSTOM, provide an array of custom button configurations here.

ButtonEvent

Interface representing a button event.


button
Type: ButtonConfig

The configuration object representing the button.
image
Type: Image | null

The currently displayed Image object.
action
Type: Action

It's the enum Action
galleryId
Type: number

Unique id of the current instance of modal, carousel, or plain galleries.

ButtonConfig

Interface representing a button configuration.


className
Type: stringoptional

Specifies the CSS class name for the button icon (e.g., with Font Awesome: 'fa fa-close').
size
Type: Sizeoptional

Specifies the button dimensions. If using a font icon, refer to the fontSize property instead. Defaults to width: 30px and height: auto.
fontSize
Type: stringoptional

Specifies the font size (CSS font-size property) when a font icon is used (e.g., with Font Awesome).
type
Type: ButtonType

Enum specifying the button type. Use ButtonType.CUSTOM to define a fully custom button.
title
Type: stringoptional

Optional title text. It sets 'title' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property
arialLabel
Type: stringoptional

Optional aria label text. It sets 'aria-label' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property
extUrlInNewTab
Type: booleanoptional

When true, opens the URL specified in the extUrl property in a new browser tab.

ButtonsStrategy

Enum controlling button visibility. Possible values:


  • DEFAULT: only close button
  • SIMPLE: close and download button
  • ADVANCED: close, download and exturl (to open a link in the current tab)
  • FULL: close, download, exturl (to open a link in the current tab), delete and full-screen
  • CUSTOM: to add an array of ButtonsConfig

ButtonType

Enum representing the button type. Possible values:


  • DELETE: delete button
  • EXTURL: extUrl button (to open a link in the current tab)
  • DOWNLOAD: download button
  • CLOSE: close button
  • CUSTOM: a fully custom button with user-defined behavior, icons, and styling
  • FULLSCREEN: full-screen button

Action

Enum representing the source of an action. Possible values:


  • NORMAL: the default action type
  • CLICK: triggered by a mouse click
  • KEYBOARD: triggered by a keyboard input (e.g., ctrl+s, cmd+s, esc, →, ←)
  • SWIPE: triggered by a swipe gesture
  • LOAD: triggered when data or images are loaded by the library
  • AUTOPLAY: triggered by autoPlay navigation
