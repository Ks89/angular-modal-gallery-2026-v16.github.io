Plain Gallery

The plain gallery displays a list of clickable thumbnails that open the modal gallery. Layouts are highly configurable, and a fully custom implementation can be provided for advanced rendering requirements.

API

To add the plain gallery to your application, include <ks-plain-gallery></ks-plain-gallery> in your component template and provide the mandatory inputs id and images.

In your TypeScript component, initialize the images array with Image instances and define the method to open the modal gallery.


To explore all customization options for the plain gallery, refer to the configuration table below. To handle plain gallery events, see the Outputs table.

For additional live demos with custom configurations, refer to the examples section of this website.


Inputs

id
Type: number
A mandatory unique id >= 0 of the current gallery.
images
Type: Image[]
The mandatory array of Image objects, that represent both the thumb gallery and the modal images. In fact, an Image is an object with both thumb and image source paths with other additional configuration parameters. For more info check Image. Avoid using this.images.push(img) to modify this array. Instead, re-assign it to trigger change detection. For example: this.images = [...this.images, newImage] creates a new array with the additional image appended.
Warning: if you need to add the same image (same URL) multiple times in the same plain gallery, you must append a random but unique query parameter to the URL to prevent caching issues. For more info, please read this issue.
config
Type: PlainLibConfigoptional

Interface to configure plain gallery.

Outputs

clickImage

A number that represents the index of the clicked image.


PlainImage

Interface that represents the optional plain image used to open the modal image when clicking on it.

If you don't provide this object, @ks89/angular-modal-gallery will scale down the modal image path to display the thumbnail.


img
Type: string

Mandatory source path of the plain image.
description
Type: stringoptional

Optional description text. To ignore this feature, don't add this property.
title
Type: stringoptional

Optional title text. It sets 'title' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.
alt
Type: stringoptional

Optional alt text. It sets 'alt' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.
arialLabel
Type: stringoptional

Optional aria label text. It sets 'aria-label' html's attribute, useful for accessibility purposes. To ignore this feature, don't add this property.

PlainGalleryConfig

Interface to configure plain gallery.


strategy
Type: PlainGalleryStrategy

Enum that represents the strategy used to show thumbnails. PlainGalleryStrategy.ROW by default.
layout
Type: PlainGalleryLayout

It can be either LineLayout and GridLayout to configure the thumbnail's layout.

PlainGalleryLayout

Empty interface implemented by both LineLayout and GridLayout.


LineLayout

Class to configure a linear plain gallery.


breakConfig
Type: BreakConfig

Object to configure length and wrap.
justify
Type: string

Sets the flexbox justify-content property. For more information, refer to the MDN documentation.
size
Type: Size

Object to change the size of thumbnails. width: 50px and height: auto by default.

GridLayout

Class to configure a grid plain gallery.


breakConfig
Type: BreakConfig

Object to configure length and wrap.
size
Type: Size

Object to change the size of thumbnails. width: 50px and height: auto by default.

BreakConfig

Interface to limit the number of items of the plain gallery or to fill other lines.


length
Type: number

Number of items to show. -1 by default.
wrap
Type: boolean

When true, images will wrap to the next line. Corresponds to the CSS flex-wrap property. Defaults to false.

AdvancedConfig

Interface for using <a> tags instead of <img> tags and for customizing the CSS background property of thumbnails in the plain gallery.


aTags
Type: boolean

When true, thumbnails are rendered as <a> tags instead of <img> tags. Defaults to false.
additionalBackground
Type: string

Sets the CSS background property of thumbnails. Defaults to '50% 50%/cover'. For more information, refer to the CSS3 background property documentation.

PlainGalleryStrategy

Enum representing the layout strategy for displaying thumbnails in the plain gallery. Possible values:


  • ROW: to show thumbnails in a row horizontally
  • COLUMN: to show thumbnails in a row vertically
  • GRID: to show thumbnails in a grid
  • CUSTOM: to show thumbnails with a custom logic
