Overview

Despite its name, @ks89/angular-modal-gallery is more than a modal gallery — new features are introduced with every major release. It consists of three main components:

  • MODAL GALLERY: available injecting the ModalGalleryService in your component
  • PLAIN GALLERY: available via <ks-plain-gallery></ks-plain-gallery>
  • CAROUSEL: available via <ks-carousel></ks-carousel>

MODAL GALLERY

The core component of this library. It displays a modal window with full-screen images, action buttons, the current image, and optionally navigation dots and previews.


  1. Current image
  2. Side previews
  3. Upper buttons
  4. Dots
  5. Previews
  6. Background (semi-transparent)
  7. Modal gallery (the main component)

For more info check Modal gallery documentation


PLAIN GALLERY

Displays a list of clickable thumbnails that open the modal gallery. Layouts are highly configurable, and a fully custom implementation can be provided for advanced rendering requirements.


For more info check Plain gallery documentation


CAROUSEL

Displays a list of scrollable images (full or fixed width) with optional previews. Images advance automatically by default. The component supports extensive configuration and allows HTML content projection into the current carousel image.


For more info check Carousel documentation
