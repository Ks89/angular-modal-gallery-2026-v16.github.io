Common

Common interfaces and classes shared across all components of this library.

Image

The primary object representing an image, including modal, thumbnail, and description data. Begin by creating an array of Image instances (using the new keyword), where each element represents a single image.


id
Type: number

Mandatory UNIQUE id to identify the image.
modal
Type: ModalImage

Mandatory interface to configure the modal image.
loading
Type: 'eager' | 'lazy'

Optional string to configure the browser's image loading strategy. Defaults to lazy.
fetchpriority
Type: 'high' | 'low' | 'auto'

Optional string to configure the image download priority (Chromium-based browsers). Defaults to auto.
plain
Type: PlainImageoptional

Optional interface to configure the thumbnail (or 'plain image').

DotsConfig

Interface to configure optional dots below the modal image.


visible
Type: boolean

When true, navigation dots are visible; when false, they are hidden. Defaults to true.

Size

Interface to customize images/icon sizes.


width
Type: string

Accepts any valid CSS width value.

Examples: '100px', 'auto', '20%'.

Default values vary by component.
height
Type: string

Accepts any valid CSS height value.

Examples: '100px', 'auto', '20%'.

Default values vary by component.

Action

Enum representing the source of an action. Possible values:


  • NORMAL: the default action type
  • CLICK: triggered by a mouse click
  • KEYBOARD: triggered by a keyboard input (e.g., ctrl+s, cmd+s, esc, →, ←)
  • SWIPE: triggered by a swipe gesture
  • LOAD: triggered when data or images are loaded by the library
  • AUTOPLAY: triggered by autoPlay navigation
