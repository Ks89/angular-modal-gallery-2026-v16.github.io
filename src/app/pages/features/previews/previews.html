Previews

The Previews component displays thumbnail previews with navigation arrows.


PreviewConfig

Interface for configuring the bottom previews in modal gallery (distinct from side-previews).


visible
Type: boolean

If false, previews will be hidden, otherwise not. true by default.
number
Type: numberoptional

Number of previews to show. 3 by default.
arrows
Type: booleanoptional

If false, the previews' navigation arrows are hidden. true by default.
clickable
Type: booleanoptional

If false, previews won't be clickable. true by default.
size
Type: Sizeoptional

Overrides the default preview size. Defaults to width: 50px and height: auto.

CarouselPreviewConfig

Interface to configure carousel previews.


visible
Type: boolean

If false, previews will be hidden, otherwise not. true by default.
number
Type: numberoptional

Number of previews to show. 4 by default.
arrows
Type: booleanoptional

If false, the previews' navigation arrows are hidden. true by default.
clickable
Type: booleanoptional

If false, previews won't be clickable. true by default.
width
Type: stringoptional

Overrides the default preview width. Defaults to 25%.
maxHeight
Type: stringoptional

Sets the maximum preview height. Overrides any larger values defined in breakpoints. Defaults to 200px.
breakpoints
Type: BreakpointsConfigoptional

Defines preview heights at different viewport widths. Defaults to xSmall: 100, small: 100, medium: 150, large: 200, xLarge: 200.

BreakpointsConfig

Interface for customizing carousel preview heights at different viewport sizes.


xSmall
Type: numberoptional

Number of pixels to be used as height for extra-small screen (defined in @angular/cdk)
small
Type: numberoptional

Number of pixels to be used as height for small screen (defined in @angular/cdk)
medium
Type: numberoptional

Number of pixels to be used as height for medium screen (defined in @angular/cdk)
large
Type: numberoptional

Number of pixels to be used as height for large screen (defined in @angular/cdk)
xLarge
Type: numberoptional

Number of pixels to be used as height for extra-large screen (defined in @angular/cdk)
